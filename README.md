# Boat Noodle POS (ก๋วยเตี๋ยวเรือ ป.ประทีป)

ระบบตัวอย่างสำหรับร้านก๋วยเตี๋ยวเรือ:
- ลูกค้าสแกน QR Code เพื่อสั่งอาหาร
- ระบบบันทึกออเดอร์
- มีหน้าเลือกวิธีชำระเงิน (จำลอง)
- มีหน้า Admin ดูรายการออเดอร์ทั้งหมด
- มีฟังก์ชันสำหรับพิมพ์บิลครัว (ตอนนี้พิมพ์ใน console; สามารถต่อกับเครื่องปริ้น ESC/POS เพิ่มได้)

## วิธีใช้งาน (บนเครื่องที่มี Python 3)

```bash
pip install -r requirements.txt
python app.py
```

จากนั้นเปิดเบราว์เซอร์ไปที่:

- หน้า QR/ลูกค้า: `http://localhost:5000/order?table=1`
- หน้า Admin ดูออเดอร์: `http://localhost:5000/admin/orders`

เวลาจะนำไปใช้จริงในร้าน:
1. เปลี่ยน `localhost` เป็น IP ของเครื่อง Mac/PC ที่รันระบบในวง Wi‑Fi ร้าน
2. เอา URL เช่น `http://192.168.1.10:5000/order?table=1` ไปสร้างเป็น QR Code แล้วแปะที่โต๊ะ
3. ลูกค้าสแกน → สั่ง → ระบบบันทึกออเดอร์ + พิมพ์บิลใน console (หรือเครื่องปริ้น ถ้าคุณต่อเพิ่ม)

คุณสามารถอัปโหลดโฟลเดอร์นี้ขึ้น GitHub ได้ทั้งชุดเลย
```
